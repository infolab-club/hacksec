Project

WayCo — стартап по удалённому заказу кофе. Перед тем, как пойти на работу/учёбы, ты можешь сделать заказ в кофейне через наш сервис к определённому времени. И когда ты подойдёшь, просто забираешь заказ без необходимости стоять в очереди или ждать его приготовления. Кроме того, есть возможность динамического определения времени прибытия в кофейню по геолокации и дорожной ситуации.

Сервис состоит из нескольких частей:
1. Web приложение для пользователей — HTML5/CSS3/JS/React
2. VK Mini App для пользователей — HTML5/CSS3/JS/React
3. iOS приложение — Swift
4. Android приложение — Java
5. Приложение для касс кофейн-партнёров для отслеживания заказов — Java
